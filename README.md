# Financial News Sentiment Monitor

A lightweight daily financial news sentiment and market volatility monitoring project.

## Project Goals
- Collect financial news every morning
- Clean and deduplicate headlines
- Generate a daily sentiment index
- Fetch market index data
- Build a daily report
- Publish results to GitHub Pages

## Current Progress
- Local project scaffold initialized
- End-to-end pipeline is running
- Daily report generation is working
- GitHub sync is working
- Source filtering and sentiment rules are under continuous improvement

## Repository Structure
- `scripts/`: runnable pipeline scripts
- `data/`: raw and processed outputs
- `reports/`: daily generated markdown reports
- `docs/`: GitHub Pages content
- `config/`: project configuration files

## Current Focus
1. Improve source quality
2. Reduce noise in headlines
3. Improve topic extraction
4. Improve GitHub Pages presentation
