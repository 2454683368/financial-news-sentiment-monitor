# Financial News Sentiment Monitor

A lightweight daily financial news sentiment and market volatility monitoring project.

## Goals
- Collect financial news every morning
- Clean and deduplicate headlines
- Generate daily sentiment index
- Fetch market data
- Build daily report
- Publish results to GitHub Pages

## Planned Modules
- news ingestion
- news cleaning
- sentiment analysis
- market data fetching
- report building
- GitHub publishing
