# 财经新闻情绪与市场波动日报系统｜简历项目表述（中文 V1）

## 项目名称
财经新闻情绪与市场波动日报系统

## 一段式简介
围绕公开财经新闻构建轻量级金融信息处理系统，完成新闻抓取、清洗去噪、情绪分析、市场指数联动、图表生成与 GitHub Pages 自动展示，用于持续观察新闻情绪变化及其与市场风险偏好的关系。

## 简历 bullet points（推荐 4 条）
- 独立搭建财经新闻情绪与市场波动日报系统，完成新闻抓取、文本清洗、情绪分析、市场数据联动、图表生成与日报发布的完整流程。
- 采用 **SnowNLP + 财经规则词典** 的混合情绪方案，针对战争、油价、通胀、加息、监管等金融语境高频误判场景进行规则修正，提升结果可解释性。
- 设计 GitHub Pages 展示页与自动化发布链路，实现日报、主题分布、情绪历史、情绪-市场联动图等内容的持续更新与线上展示。
- 在分析层加入单日情绪与 **3 日滚动情绪** 视角，结合沪深300、上证指数收益率观察新闻影响的滞后性与持续发酵特征。

- 基于 SnowNLP 与财经关键词规则构建轻量级情绪分类方案，并针对金融语境中的典型误判进行迭代优化。
