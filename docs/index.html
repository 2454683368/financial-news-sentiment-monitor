方法说明

Methodology

  • 从少量相对稳定的公开财经新闻源抓取标题。
  • 通过规则过滤、去重、域名排除等方式清洗标题。
  • 情绪评分采用 SnowNLP + 财经关键词规则 的混合方案。
  • 市场联动不仅看单日情绪，也关注近 3 日滚动情绪，因为新闻影响常具有滞后性和持续发酵效应。
  • 随着历史样本积累，多日联动分析会越来越有信息量。
  • Headlines are collected from a small set of relatively stable public financial sources.
  • Titles are cleaned with rule-based filtering, deduplication, and domain exclusion.
  • Sentiment uses a hybrid score: SnowNLP + finance-specific keyword rules.
  • Market linkage looks beyond same-day sentiment and also tracks a 3-day rolling sentiment view, since news effects often have lag and persistence.
  • Historical linkage becomes more informative as daily samples accumulate.

流程

  1. Ingest daily financial headlines

  4. Fetch market index data
  5. Generate charts, reports, and GitHub Pages output

已知局限

Known Limitations

  • 仍可能存在少量边界噪声
  • 混合情绪评分仍不是完美分类器

  • Historical linkage is still early because the sample is short
